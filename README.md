# PythonCalculator
A simple GUI Python Calculator using the tkinter library 
and math library for trigo function

## Features:
- Resizable
- 4 operators
  - Addition `[+]`
  - Subtraction `[-]`
  - Multiplication `[ร]`
  - Division `[รท]`
- Error detection
  - Errors
  - Division by Zero Errors
  - Name Errors
- Clear All button
- Clear button
- Decimals
- Trigonometry
- Ability to type in expressions
