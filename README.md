# PythonCalculator
A simple GUI Python Calculator using the tkinter library (`from tkinter import *`)

## Features:
- Resizable
- 4 operators
  - Addition `[+]`
  - Subtraction `[-]`
  - Multiplication `[ร]`
  - Division `[รท]`
- Error detection
  - Errors
  - Division by Zero Errors
  - Name Errors
- Clear button
- Decimals
- Ability to type in expressions
